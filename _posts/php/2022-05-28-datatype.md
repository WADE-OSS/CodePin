# 기본 타입
타입(data type)이란 프로그램에서 다룰 수 있는 값의 종류를 의미합니다.
여러 가지 형태의 타입을 미리 정의하여 제공하고 있으며, 이것을 기본 타입이라고 합니다.

제공하는 기본 타입은 다음과 같습니다.
1. 불리언(boolean)
2. 정수(integer)
3. 실수(float)
4. 문자열(string)
5. 배열(array)
6. 객체(object)
7. 리소스(resource)
8. NULL
